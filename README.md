## Instalation

To start ths project you have to make this steps:

1. Clone down this repository. You will need use npm.

2. Installation:

$ npm install

3. To Run Test Suite:

$ npm test

4. To Start Server:

$ npm start

5. To Visit App:

localhost:3000
